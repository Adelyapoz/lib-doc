
Подключение в фронту

1. Должен быть скачан файл local.jar
2. Открыть файл local.jar (например winrar)
3. Внутри local.jar найти файл local.conf (открыть блокнотом)
4. Найти строку db = { url = "jdbc:postgresql:kapinuss",  login = "postgres", password =  }
5. После password = поставить свой пароль, такой же как в базе данных (например, password = 123)
6. Сохранить изменения
7. База данных должны иметь название kapinuss (если другое - то надо переименовать)
8. Кликнуть на файл local.jar правой кнопкой мыши и нажать Открыть в терминале
9. В открывшемся терминале ввести java -jar "C:\Program Files\Jar\local.jar";  (Должен быть свой путь до файла)
10. Должен быть скачан node.js
11. Должен быть склонирован из репозитория lib-front
12. Открыть терминал
13. Ввести путь до lib-front (например, C:\Users\admin\lib-front>)
14. Ввести npm start
