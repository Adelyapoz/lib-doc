Rer01

Название:
Проверка регистрации пользователя (позитивная)

Шаги
1. Зайти на сайт http://localhost:3000/
2. Нажать кнопку "Login" в правом верхнем углу  
3. Ниже поля ввода логина и пароля нажать кнопку "Don't have an account? Sing Up"                                   
4. Ввести данные:
    login - vasa
    First Name -vasa
    Surname - vasa
    Email - vasa@mail.ru
    Password -Vasa1234 (Пароль должен содержать одну верхнюю букву, нижнюю букву, цифры и минимальную длину 8)

Ожидаемый результат:
1. Появляется окно уведомляющее об успешной регистрации
