Название: Добавление нового пользователя через API

Номер: A-001

Шаги:
1. Открыть Postman.
2. Выбрать метод запроса: POST.
3. В поле URL ввести: `http://localhost:8080/reg`.
4. В разделе "Body" выбрать опцию "raw" и установить формат "JSON (application/json)".
5. Вставить следующий JSON-запрос в поле Body:
{
  "id": "3",
  "login": "erty",
  "password": "34525zcxv",
  "firstname": "vsgdfgzxv",
  "surname": "Bsdfgasdfgcvz",
  "email": "xzavvvasv@gmail.com"
}
6. Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 200 Ok
В теле ответа приходит значение "1", что указывает на успешную регистрацию пользователя.
