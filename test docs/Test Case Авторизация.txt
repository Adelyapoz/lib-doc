Авторизация
Предусловия:
+ Установлено подключение к сайту http://localhost:3000/
Действия: 
1. В правом верхнем углну нажать на кнопку "Login" - ОР Открывается страница с полями воода Login и Password
2. Вводим валидные данные (например login -Neewbie, password - 123) соблюдая регистр - ОР  Пользователь авторизован, открывается начальная страница сайта, в правом верхнем углу указан Логин пользователя
